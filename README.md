# 3S Dashboard
データベースへのアクセス機能付きダッシュボードクライアント。

# Environment
- Dashboard
  - Vue.js 3.3.4
  - Vite 4.4.9
  - Tailwind CSS
  - Vuestic UI 3
  
- Website
  - Vue.js 3.3.4
  - Vite 4.4.9
  - Tailwind CSS


# StartUp
1. リポジトリをクローン
```sh
git clone https://github.com/2023-3Skoreisai/FrontEnd.git
```

2. クローンしたディレクトリに移動
```sh
cd FrontEnd
```

3. 依存関係をインストール
```sh
npm i
```

4. 開発サーバーを起動
```sh
npm run dev
```

Viteの構成は次の通りです。
- 開発サーバー  
`npm run dev`
- ビルド  
`npm run build`
- buildのプレビュー  
`npm run preview`
- esLint  
`npm run lint`
